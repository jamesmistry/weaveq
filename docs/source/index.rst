JiggleQ
=======
A tool that pivots and joins across collections of Python objects, with special support for pivoting and joining Elasticsearch results.

JiggleQ exposes both a simple command-line interface for use as a standalone application, as well as a programmatic interface for use from Python code.

Compatibility
-------------

JiggleQ requires Python 2.7.6 or above.

JiggleQ is compatible with all versions of the Elasticsearch Python DSL library.

Example Usage
-------------

jiggle -q "select asdsa "
